# 克隆挑取补偿精准度风险分析与排查方案

## 1. 当前问题判断

当前功能流程已经基本跑通，剩余核心风险是：检测到的目标细胞克隆是否能被精准补偿到视野中心或挑取中心。

结合现象分析，主要风险来自两类误差：

1. 机械传动误差：丝杆间隙、反向空程、重复定位误差。
2. 标定参数误差：视野尺寸、脉冲/mm、图像坐标到平台坐标的方向与比例换算不准。

这两类误差都会直接影响 `workflow/compensate_executor.py` 中的补偿计算。

当前补偿核心逻辑可以理解为：

```text
图像中目标相对中心的像素偏移
  -> 通过 mm_per_pixel 换算成毫米偏移
  -> 通过 pulses_per_mm 换算成电机脉冲
  -> 根据方向符号计算目标平台坐标
  -> 控制 XY 平台移动
